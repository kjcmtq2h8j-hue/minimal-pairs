Export Recording Session

Share a link with a native speaker so they can record all {{ word_count }} words in their browser — no software needed.

1

Test locally

Open this link on your own computer to preview the recorder before sharing.

Open ↗
2

Host the recorder page (one-time setup)

The native speaker's browser needs HTTPS to access the microphone, so the recorder page must be on the internet. Choose a free option below — you only do this once.

Netlify Drop ✓ Easiest

Drag the recorder/index.html file onto app.netlify.com/drop. Get a live HTTPS URL in seconds.

GitHub Pages

Create a repo, upload index.html to it, enable Pages in Settings → Pages. Gives a stable URL like yourname.github.io/recorder.

The file to upload is at: minimal_pairs/static/recorder/index.html

3

Build the shareable link

Paste your hosted URL below — the session data will be appended automatically.

4

Send the link to the speaker

Share via WhatsApp, email, or any messaging app. The speaker opens the link, enters their name, records each word one by one, then downloads a zip file of their recordings to send back to you.

📦 The session data ({{ word_count }} words) is embedded in the URL itself — no server or database is involved on the speaker's side.
5

Import their recordings

Once the speaker sends back the zip file, import it here to attach all recordings to your pack.

⬆ Go to Import Recordings
